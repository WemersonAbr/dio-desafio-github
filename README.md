# Desafio sobre Git/GitHub da DIO

## Alguns comandos de git para usar no git bash ou cmd

  git init - inicia o git para fazer o versionamento do código;
  git config --global user.email ou user.name " " - esees dois comandos é feitos para que quando fizermos commits eles tenham um autor;
  is -a - esse comando lista pastas ocultas e oque tem dentro delas;
  mv - move uma pasta ou arquivo;
  git add * - adiciona a modificação feita no arquivo local para o git;
  git commit -m " " - adiciona o comentário de oque foi feito de alteração no arquivo;
  git push - "empurra" o arquivo para o github com as alterações feitas;
  git pull - "puxa" o arquivo do github para a maquina local;
  ctrl L - limpa o terminal do git bash.

## Alguns comandos no CMD (windows)

  dir - lista todos os diretórios que o usuário está utilizando;
  cd / - navegar entre as pastas, utilizando a / você se move para a base do diretório;
  cd .. - sai de uma pasta;
  CLS - limpa o terminal;
  TAB ( do teclado ) - serve para completar alguma escrita;
  mkdir - cria uma pasta;
  echo - manda uma mensagem no terminal ( como se fosse um print em python ou console.log em js );
  del (teclado) + algum nome de pasta - deleta oque tem debtro da pasta;
  rmdir + nome da pasta + /S /Q - exclui a pasta com tudo que tem dentro dela.
